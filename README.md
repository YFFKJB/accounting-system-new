# accounting-system
工作室记账系统

## 项目简介
这是一个简单易用的在线记账系统，支持多用户管理各自的收支记录。系统采用Web方式实现，使用Vercel Serverless部署。

## 技术架构
- 前端：HTML + CSS + JavaScript
- 后端：Vercel Serverless Functions
- 数据库：MongoDB Atlas
- 部署：Vercel

## 主要功能
1. 用户管理
   - 用户注册和登录
   - 密码加密存储
   - JWT token认证

2. 记账功能
   - 添加收入/支出记录
   - 记录包含：日期、类型、金额、分类、备注
   - 查看历史记录
   - 按月份统计收支情况
